🌿 WellnessFlow – Smart Habit & Wellness Tracker

WellnessFlow is a comprehensive Android wellness application designed to help users build healthy habits, track emotional well-being, and maintain consistent daily routines.

Developed as part of IT2010 – Mobile Application Development (Lab Exam 03) for the BSc (Hons) in Information Technology – 2nd Year, this project demonstrates advanced Android development concepts, clean architecture, and professional UI/UX design.

🎯 Project Overview

WellnessFlow combines multiple wellness-focused tools into one unified platform:

📊 Habit tracking with per-day progress

😊 Mood journaling with trend visualization

💧 Smart hydration reminder system

📈 Advanced analytics & statistics

🧩 Interactive relaxation mini-game

🏠 Home-screen widget integration

🎨 Theme customization & persistent settings

The application emphasizes user engagement, data persistence, and responsive UI design.
